Linux CLI Cheat-Sheet (Rahul’s List Explained)

#CommandWhat it does / why you care
1pwdPrint absolute path of present working directory.
2lsList names of files & dirs (alphabetical).
3ls -RRecurse into sub-dirs and list everything.
4ls -tSort by modification time, newest first.
5ls -lLong format: perms, owner, size, date, name.
6ls -ltLong listing sorted by time.
8ls -laLong + show hidden files (dot-files).
9ls -lRaLong, recursive, all files (hidden too).
10ls -lrReverse whatever order is currently used.
11ls -sShow allocated blocks next to each name.
12ls *.jsList only JavaScript files.
13ls Zoo*List files whose names start with “Zoo”.
14ls ..List parent directory without moving.
15cdChange directory (home dir if no arg).
16cd ..Go up one level.
17cd ../../Go up two levels.
18touch fCreate empty file or update its timestamp.
19cat fDump whole file to terminal.
20cat > fType text; Ctrl-D saves & overwrites file.
21cat >> fAppend typed text to file.
22mkdir dCreate directory “d”.
23mkdir d && cd dCreate and step inside in one go.
24mkdir -p a/b/cCreate nested directory path.
25mv old newRename or move file/dir.
26mv path/old path/newSame as 25 with full paths.
27cp f newCopy file.
28cp -r d newCopy directory recursively.
29rm fDelete file (irreversible).
30rm -r dDelete directory and everything inside.
31chmod ugo-rwx fRemove all permissions from everybody.
31bchmod -R ugo-rwx dSame, recursive for directory.
32chmod u+x fGive owner execute permission.
33chmod g+wx fGive group write & execute.
34chmod u-x fRemove owner execute.
35chmod 664 fNumeric: rw-rw-r--.
36echo 'msg'Print message to terminal.
37head fFirst 10 lines.
38tail fLast 10 lines.
39head -20 fFirst 20 lines.
40tail -n +25 f | head -5Lines 25-29 only.
41wc fLines, words, characters count.
42grep "pat" fShow lines containing pattern.
43grep "pat" f | wc -lCount matching lines.
44grep -c "pat" fSame as 43, built-in count.
45grep -h "pat" fSuppress file name prefix.
46grep -hi "pat" fCase-insensitive + -h.
47grep -hir "pat" dRecursive, case-insensitive search.
48grep -hin "pat" fShow line numbers too.
49grep -hinw "pat" fWhole-word, case-insensitive, numbered.
50grep -o "pat" fPrint only the matched text.
51grep -w "pat" fWhole-word match.
52historyList previous commands.
53bash scriptRun file as Bash program.
54grep "ERROR" logAll error lines.
55grep -v "INFO" logInvert: skip INFO lines.
56grep -A 5 ERROR logError line + 5 after.
56bgrep -B 5 ERROR log5 lines before error.
56cgrep -C 5 ERROR log5 lines before & after.
57sed -n '/ERROR/ p' fPrint only ERROR lines.
58sed 's/ERROR/CRITICAL/' fReplace first ERROR per line.
59sed -ibackup 's/E/C/' fIn-place edit with backup.
60sed '3 s/E/C/' fReplace only on line 3.
60bsed '3,5 s/E/C/' fReplace on lines 3-5.
60csed -n '3,/ERROR/ p' fPrint from line 3 to first ERROR.
61awk '/ERROR/{print $0}' fPrint ERROR lines.
62awk '{gsub(/E/,"C"); print}' fGlobal replace E→C.
63awk 'BEGIN{print "HEADER"} {print} END{print "FOOTER"}' fAdd header & footer.
64awk '{print $1, $2}' fFirst two columns.
65awk -F"," '{print $1, $2}' fCSV: first two fields.
66awk '{count[$2]++} END{print count["ERROR"]}' fCount occurrences in 2nd column.
67awk '$1>1598863888' logRows whose first column > epoch.
